About

Hi, I'm Ellen, a software developer in Cambridge, England. I used to be a historian.

Occassionally I write up things I've been working on or thinking about or want to remember. Usually there's code.

The blog's theme varies according to the user agent's prefers-color-scheme setting. Pages with example code use PrismJS for syntax highlighting. If JavaScript is blocked, default CSS tries to provide a graceful fallback.

A few side projects

  • poetry-scaffold-plugin installs and configures some common Python dev dependencies, which I got tired of setting up for each project.
  • Django Hospital is a way to practice writing Django queries. I wrote it a few years ago to help some junior developers learn about the Django ORM. Occassionally I upgrade the dependencies.
