About

Hi, I'm Ellen, a software developer in Cambridge, England. I used to be a historian.

This blog is where I write up things I've been working on or thinking about. Usually there's code.

The blog's theme varies according to the user agent's prefers-color-scheme setting. Pages with example code use PrismJS for syntax highlighting. If JavaScript is blocked, default CSS tries to provide a graceful fallback.
